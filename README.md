# Who follows me?
Check out who follows you, but you're not following back and vice versa.

This webapp is hosted [here](https://twitter.michielborkent.nl). 

## Configure
Copy `config.example.edn` to `~/.wfm/config.edn` and fill in the blanks.

## Run locally / develop

    lein repl
    (dev)
